# BluePresence

Blue Archiveの起動を検出し、Discord Rich Presenceに名前、フレンドコード、現在の起動からの経過時間を表示するWindows常駐アプリです。

## 使い方

1. ダウンロードしたBluePresence.exeを起動してください。設定ファイルがない初回だけ設定画面が開きます。
2. 名前と必要ならフレンドコードを入力して保存を押してください。要らないならスキップしてください。
3. Discordデスクトップアプリを起動した状態でBlue Archiveを起動します。
4. 初回起動以外は設定画面が勝手に開かれません、通知領域のBluePresenceアイコンをダブルクリックか右クリックをして設定を押すと設定を開けます。
5. 完全に終了するには通知領域メニューで右クリックをして終了を押してください。

設定は `%USERPROFILE%\AppData\LocalLow\Y2KDevs\BluePresence\config.json` に保存されます。

## note
- 表示時間はBluePresenceがゲームを検出した時点からの経過時間です。過去の累計プレイ時間ではありません。
- Discordデスクトップアプリが停止中の場合は接続できるまで自動再試行します。
- 名前とフレンドコードは、設定されている項目だけDiscordに表示します。
